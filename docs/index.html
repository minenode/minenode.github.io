MineNode

MineNode is a Minecraft: Java Edition server built on top of Node.JS

Built from the ground-up as a performant and easily modifiable Minecraft server.

Why Node.JS?

Node.JS, combined with native C++ modules and TypeScript provide for a scalable and stable codebase while providing stable built-in standard library features for networking and other low-level abstraction.

The use of Node.JS also brings with it the opportunity for high-performance scripting for plugins, as well as allowing plugin developers to use the extensive NPM package collection.

C++ native modules are used to bring performance to CPU-intensive and low-level processes where needed.
